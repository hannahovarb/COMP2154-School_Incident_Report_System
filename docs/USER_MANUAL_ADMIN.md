## Getting Started

### Access Requirements
- Administrator account (assigned by system administrator)
- Secure login credentials
- Internet connection

### First Time Login
1. Navigate to the application URL
2. Enter your admin email and password
3. You'll be redirected to the Admin Dashboard
4. **Important**: Change your password immediately if using default credentials
